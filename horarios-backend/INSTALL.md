# INSTALL

## 1) Instalar las Node dependencies

```bash
$ npm install
```

## 2) Database

- Eliminar (por si fuera necesario)

```bash
$ npm run dropdb
```

- Crear 

```bash
$ npm run createdb
```

- Crear los modelos (tablas) 

```bash
$ npm run modelos
```

## 3) Correr el backend (server)

- modo desarrollo

```bash
$ npm run dev
```
